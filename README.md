# Library Manager

It is an application to manage library (public or at home).

## Screenshots
![Screenshot1](_Screenshots/main_screen.jpg)  

## Features:
- Books database
- Lending history
- Users/clients database
- Search client/book
- Logging (support for multiple accounts with roles: administrator, employee)
